AusRegistry's EPP Client Library for the Java Programming Language.

* Removed all ccTLD and gTLD specific functionality from the toolkit. These features are now
modularised into separate branches for gTLD and ccTLD.

Changes in version 2.0.6.12
* Added sunrise application extension support for domain delete application based on draft
specification for application.xsd.

Changes in version 2.0.6.11
* Added idn-1.0 extension. Added implementation to for IDN extension for domain create and domain info response 
based on the draft 

Changes in version 2.0.6.10
* Update sunrise application extension implementation to match changes to draft specification around statuses.

Changes in version 2.0.6.8
* Added sunrise application extension support for domain create application and domain info application based on draft
specification for application.xsd.

Changes in version 2.0.6.7
* Added equals methods to DSData and KeyData. Made KeyData not use signed short and byte logic.

Changes in version 2.0.6.6

* Implemented support for hosts attribute of name element in domain info command. Previous versions defaulted to "all".
